what is this ?

DUGL the DOS Ultimate Game Library, is a DOS only 32bits Game and Graphics library.
It aims to give a modern and the fastest graphic software rendering library on DOS,
and maybe ever :)

License ?

See License.txt

Requirement ?

The DJGPP 2.04 C/C++ compiler. You can get it from 'www.delorie.com/djgpp'.
A Dos compatible operating system Including FreeDOS, DrDOS, MsDOS and any MsWin9x.
To be able to run your apps builded with DUGL you will need :
- A Pentium MMX or 100% compatible CPU (recommended a PII 300Mhz+)
- Not less than 4MB of memory (not tested with this amount)
- A VESA 2.0 compliant graphic card with at least 1MB of VRAM

How to install ?

1/ Copy DGraph.h, DControl.h, DIPX.h, DUGL.h to %DJGPP_PATH%/include/DUGL
2/ Copy libdugl.a %DJGPP_PATH%/libs
